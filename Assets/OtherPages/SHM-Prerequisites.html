A quick revision of the building blocks of geometry

Of course we are taking the Euclidean view and not the modern set theoretic view
Point: A point is that which has position but no magnitude.


Basically what it means is that the only property that a point has is its position. It has no property called length or area or volume – it only has position (similar to the case that a 'color' has no taste, it has only the property of 'color' like green, red etc.;
or like the case that a 'sunset' has no mass, it can have time or brightness values but it is meaningless to ask for the mass of a sunset).

Try to zoom in on the point shown in the next slide as a blue circle or dot. You will find that although you are zooming in, the size of the dot does not change because it is a point and it is not supposed to have a 'size'. the dot is used for our convenience to locate the point, because an actual theoretical point will not be visible to us.

You will also notice numbers associated with the point given in brackets. The position of a single entity in an infinite universe is again meaningless (try to imagine a point in empty space, there is no point in saying that it is here or it is there, for we cannot say anything about its position because we have no landmark/marker to compare it with). Thus, if we need to specify position of a point, we have to specify it with respect to a landmark i.e. another point which we take to be fixed for all time unlike our original point which can move. This fixed point or landmark is the origin.
